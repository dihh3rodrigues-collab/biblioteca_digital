# Sistema de Gerenciamento de Biblioteca Digital

## Objetivo
Desenvolver um sistema em Python para auxiliar bibliotecários na gestão de documentos digitais.

## Funcionalidades

- Listar documentos
- Adicionar documentos
- Renomear documentos
- Remover documentos

## Tecnologias Utilizadas

- Python
- Git
- GitHub

## Como Executar

1. Abrir o terminal.
2. Navegar até a pasta do projeto.
3. Executar:

python main.py
